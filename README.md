# Notes

This is a line following robot. In the programme, the robot move very slowly. The robot will continuously adjust one of its two sensors to be tracking on black line.

## How to detect black line

The code provided in Maqueen v4 extension will return 0 if the sensor detects a black line

```
maqueen.read_patrol(maqueen.Patrol.PATROL_LEFT) == 1  # 1 indicates white line
maqueen.read_patrol(maqueen.Patrol.PATROL_RIGHT) == 1  # 1 indicates white line
```

## Requirements

MakeCode Extension required to run this:
- Maqueen v4

The robot setup:
- Microbit v1, with firmware interface version: 0249
- DFRobot micro:Maqueen v4.0

Line map:
- About 1.5 cm wide
- Rounded corners
- Black line
